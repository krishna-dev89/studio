# **App Name**: CybersafeIndia

## Core Features:

- Home Page: Introduction to CybersafeIndia, its mission, and quick links to key sections.
- Cyber Threat Education: Sections detailing common cyber threats like phishing and identity theft, with clear explanations and icons.
- Safe Digital Habits Guides: Guides on password safety, social media safety, and safe online transactions.
- AI-Powered Threat Explanation: An AI tool that simplifies and explains future threats like AI-based attacks and deepfakes for users of all backgrounds.
- Reporting Guidance: Step-by-step guidance on how to report cybercrime, with links to official reporting portals.
- Knowledge Quizzes: Interactive quizzes to test user knowledge and understanding of cybersecurity concepts.
- Newsletter Signup: A newsletter signup form to keep users updated on the latest threats and safety measures.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to inspire trust and security, reflective of a government initiative.
- Background color: Light blue (#D3E2FF), a desaturated shade of the primary, for a calm, trustworthy backdrop.
- Accent color: Indigo (#4529FF) for highlights and calls to action; contrasting saturation and brightness make it distinct from the primary color.
- Headline font: 'Space Grotesk' sans-serif, with a modern tech feel for headlines.
- Body font: 'Inter' sans-serif, for clean, readable body text.
- Use lock, shield, and network-like icons to represent security concepts.
- Mobile-friendly, responsive layout to ensure accessibility across all devices.
